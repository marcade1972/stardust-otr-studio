STARDUST BROADCAST 1.9.3.1

Install
1. In WordPress, open Appearance > Themes > Add New > Upload Theme.
2. Upload this ZIP and activate it.
3. Open Settings > Permalinks and click Save Changes once so the /actors/ address is registered.

Actor Database
- Open Actors in the WordPress dashboard.
- Choose Add New Actor.
- Enter the actor's name as the title.
- Add the full biography in the main editor.
- Set a portrait as the Featured Image.
- Complete the Actor Details panel.
- The public directory is available at /actors/.

Colors
Open Appearance > Customize > Stardust Station Settings to change the Primary Navy and Accent Gold colors.

Favicon
The theme includes shooting-star and microphone icons. WordPress Site Icon settings can still override them.

Broadcast Library (1.7.7)
-------------------------
Assign the “Broadcast Library” template to the WordPress page with the broadcast-library slug. The page displays all current curated series libraries with live search and genre filters. Future series can be added to the template configuration or through the stardust_broadcast_library_series filter.

Version 1.7.9.1 setup:
- Edit the Suspense WordPress page and select the "Suspense Episode Library" template.
- The template reads MP3 files recursively from https://stardustotr.com/suspense/.
- Set a Featured Image on each series page to use custom artwork on the Broadcast Library card. When no Featured Image is set, the theme uses a matching genre image.
- The Broadcast Library now groups series into genre collections automatically from its reusable series configuration.
